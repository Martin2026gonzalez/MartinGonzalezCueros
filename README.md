# Martin Gonzalez — Marroquineria & Calzado

Sitio web institucional para **Martin Gonzalez**, artesano de marroquineria y calzado con sede en **Gualeguay, Entre Rios, Argentina**.

## Descripcion

Landing page de presentacion del negocio, con catalogo de productos, informacion de contacto y paginas legales.

## Estructura del proyecto

```
Martingonzales/
├── Index.html                    # Pagina principal (landing page)
├── boton-arrepentimiento.html    # Pagina: boton de arrepentimiento
├── politica-privacidad.html      # Pagina: politica de privacidad
├── terminos-condiciones.html     # Pagina: terminos y condiciones
├── imagenes/                     # Assets de imagenes del catalogo
│   ├── logo.webp
│   ├── carteras.webp
│   ├── Llaveros.webp
│   ├── cintos Clasico Hombre.webp
│   ├── cintos Clasico Mujer.webp
│   ├── cintos Clasico Hebillas1.webp
│   ├── cintos Clasico Hebillas2.webp
│   └── Borcegues de dama.webp
└── package.json
```

## Tecnologias

- HTML5 / CSS3 / JavaScript vanilla
- Google Fonts (Playfair Display, Cormorant Garamond, Raleway)
- Sharp (optimizacion de imagenes — devDependency)

## Productos destacados

- Carteras artesanales
- Llaveros de cuero
- Cintos clasicos (hombre y mujer)
- Borcegues de dama
